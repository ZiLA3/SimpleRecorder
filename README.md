# SimpleRecorder
Simple WebCam Recorder with python

### Functions ###
- Mirrored Cam (hflip)
- Record Video
- Simple Effect (Negative image)

### Controls ###
- Space: Record Start/Stop
- Tab: Convert Negative/Positive
- ESC: Exit

### Notice ###
- When Recording, red dot appears in topleft.
- Recorded Video is saved as "Date.avi".
